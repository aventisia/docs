# Overview

**Worc-space** is a platform for managing automation workflows, AI models, support tickets, and orchestrator services like machines, queues, and executions. It is designed for building, deploying, and managing automation agents and AI solutions in a centralized environment.

---

- **Main Menu (Left Panel):** Quick access to:
  - Agents
  - AI Models
  - Support
  - Orchestrator tools:
    - Machines
    - Queues
    - Triggers
    - Executions
    - Key Store
    - Vault
- **Top Search Bar:** For searching flows or agents
- **Create Agent Button:** To add new automation agents
- **Pagination Controls:** Navigate through multiple pages if more agents exist
