# Run VBScript

The **Run VBScript** node allows you to execute small scripts written in the VBScript language. This node is perfect for situations where you need to perform automated desktop tasks, manipulate data, or interact with the Windows operating system using VBScript code.

Typical use cases include:

- Automating repetitive desktop processes through scripting.
- Accessing system functions via VBScript.
- Performing complex tasks that might require scripting power.

{/* ![Run VBScript](./run-vbscript.png) */}

## Configuration Options

| Field Name        | Description                   | Input Type | Required? | Default Value |
| ----------------- | ----------------------------- | ---------- | --------- | ------------- |
| **VBScript Code** | The VBScript code to execute. | Text       | Yes       | _(empty)_     |

## Expected Output Format

The output of the **Run VBScript** node is textual data, typically the result or any messages from executing the script, available in the form of **ScriptOutput**.

- Text output may include success messages, error information, or specific data retrieved by the script.

## Step-by-Step Guide

<Steps>
### Step 1

Add the **Run VBScript** node to your flow.

### Step 2

In the **VBScript Code** field, type or paste the VBScript code you wish to execute.

### Step 3

After executing, retrieve the output from the **ScriptOutput** field for use in subsequent nodes.

</Steps>

<Callout type="info" title="Tip">
  Ensure your VBScript code is correctly formatted and error-free to prevent
  execution issues.
</Callout>

## Input/Output Examples

| Example Description                    | VBScript Code                     | Output Value  | Output Type |
| -------------------------------------- | --------------------------------- | ------------- | ----------- |
| Display a message box                  | `MsgBox "Hello, World!"`          | Hello, World! | String      |
| Simple math operation returning result | `result = 5 + 10 : MsgBox result` | 15            | String      |

## Common Mistakes & Troubleshooting

| Problem                            | Solution                                                                                                                                 |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **VBScript Code fails to execute** | Double-check the syntax of your VBScript. Ensure there are no missing semicolons or unmatched quotes.                                    |
| **Unexpected script output**       | Verify your VBScript logic to ensure it performs the intended tasks. Use built-in VBScript debugging methods to troubleshoot the script. |
| **ScriptOutput appears empty**     | Make sure your script includes output instructions like `MsgBox` or print statements to generate visible results in **ScriptOutput**.    |

## Real-World Use Cases

- **Automate Email Notifications**: Write a VBScript to automatically send desktop notifications or emails when specific conditions are met.
- **System Interactions**: Use VBScript to manage files, deploy applications, or perform batch operations directly in Windows.
- **Data Processing**: Automate data transformations or calculations that are complex to achieve using simple rules.
