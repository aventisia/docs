## Convert JSON to Custom Object

This form allows users to input a JSON string and convert it into a custom object or list structure, suitable for further automation or data manipulation tasks.

![alt text](../../../assets/variable/convert-JSON-to-custom-object.png)
---

### JSON Input

#### 1. Field Name- `JSON Data`  
- **Purpose:** The JSON string that will be converted to a custom object.

#### 2. Field Name- `Target Type`  
- **Purpose:** Specifies the type of the custom object to deserialize the JSON into.

#### 3. Field Name- `Outputs: CustomObject`  
- **Purpose:** Returns the resulting custom object after JSON deserialization.

---
- **Example:**
  ```json
  [
    { "id": 1, "name": "Alice" },
    { "id": 2, "name": "Bob" }
  ]

### Use Case

Use this tool when:
- You need to parse raw JSON data into a structured format.
- You're integrating APIs or external data sources into your workflow.
- You want to easily access fields and iterate over JSON data programmatically.
