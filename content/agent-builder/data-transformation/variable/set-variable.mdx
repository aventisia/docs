# Set Variable

## Description

This operation assigns a specific value to a variable. It is commonly used to store data dynamically within workflows or automation processes.

![alt text](../../../assets/variable/set-variable.png)

## Input

- **Variable Name**: The name of the variable that will be assigned a value.  
  - Example: `var_CountryName`

- **Value To Set**: The data or information to be stored in the specified variable.  
  - Example: `Mexico`

## Output

- The variable (`var_CountryName`) will hold the assigned value (`Mexico`), making it available for further use in subsequent operations.
